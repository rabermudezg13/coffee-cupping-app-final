# ☕ Coffee Cupping App

Professional coffee evaluation and cupping application.

## Features

✅ **Coffee Bag Evaluation System**
- ⭐ Star rating (1-5 stars)
- 🌍 Origin and producer tracking
- 💰 Cost analysis
- 🔥 Roast level and date
- ☕ Preparation method
- 👃 Aroma evaluation (dry/wet)
- 🎨 Flavor notes
- 💭 Recommendations

✅ **User Management**
- Secure login system
- Personal dashboard
- Review history

✅ **Professional Design**
- Clean, responsive interface
- SCA-inspired layout
- Mobile-friendly
